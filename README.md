# Prospect Intelligence

Prospect Intelligence is a Windows desktop application for baseball prospect analysis and sports card market research.

## Features

- Minor league player performance tracking
- Promotion Score
- Superstar Score
- Buzz Score and prospect news tracking
- Historical sports card sales analysis
- Median and average sale prices
- Sales volume and pricing trends
- Historical card price charts

## eBay Marketplace Integration

Prospect Intelligence is being developed to use the eBay Marketplace Insights API to retrieve completed sales for selected sports cards.

Completed sales are filtered to remove non-comparable listings such as graded cards, parallels, lots, and unrelated variations. The resulting data is used for local historical analysis, pricing charts, and market trend calculations.

eBay listing information is not republished or redistributed.

## Development Status

Prospect Intelligence is currently in private development and testing.
